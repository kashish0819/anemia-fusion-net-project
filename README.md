# 🩸 AnemiaFusionNet
An AI-powered multimodal anemia detection system that combines retinal image analysis, blood parameters, and regional health intelligence to provide fast, accurate, and intelligent anemia screening.

---

## 🚀 Features

- 👁️ Retinal Image-Based AI Detection
- 🩸 Blood Parameter Analysis (Hb, MCV, MCH, MCHC)
- 🌍 Regional Health Intelligence
- 🤖 Multimodal FusionNet AI Model
- 📊 Analytics Dashboard
- 📄 Professional PDF Medical Reports
- 📁 Prediction History & Report Download
- 📱 Responsive Healthcare Web Interface
